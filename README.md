# El Diccionario Comunitario de Python 🐍📚

Este es un proyecto colaborativo diseñado para que estudiantes aprendan el flujo de trabajo en proyectos de Código Abierto (Open Source). El objetivo es construir un diccionario de términos técnicos de Python de forma comunitaria.

## 🎯 Objetivo del Taller

Aprender y practicar el flujo de trabajo profesional utilizando Git y GitHub:
1. **Forking**: Crear una copia propia del repositorio.
2. **Clonación**: Descargar el proyecto a tu máquina local.
3. **Branching (GitFlow)**: Crear ramas para nuevas características.
4. **Pull Requests (PR)**: Proponer tus cambios al repositorio central.
5. **Code Review**: Entender cómo se revisan las contribuciones.

## 🚀 Cómo ejecutar el proyecto

Para ver los términos actuales en el diccionario, simplemente ejecuta el script principal con Python:

```bash
python main.py
```

## 🛠️ Requisitos

- Python 3.x instalado.
- Git configurado en tu equipo.
- Una cuenta de GitHub.
